""双引号，可以用来引用变量，可以保存变量中的空白，如果不加就是变量替换将不保留空白
`这是反引号(1前面那个字符)，他包围的是一个可以执行
=可以用来进行赋值(赋值的时候必须紧跟变量和值 比如 a=20,等号两边不能有空格)，和比较字符串(比较字符串和前面相反)
'这是单引号，他包围的是模式与选项等
&表示追加字符
$在模式中表示最后一个字符
$?上一个命令的返回值
$$当前进程号
$!后台运行的最后一个进程的id号
$#传递进来的参数个数
$0代表脚本文件的名字
$-显示shell使用的当前选项，与set命令功能相同
$1第一个参数，大于9的参数必须用大括号括起来比如${10}
$*,$@表示位置参数，(在makefile中$@代表当前目标)
$(shell command)相当于反引号
$'\n' 表示换行(单引号)
:代表一个空命令
.点命令
%可以用来取模，与正则表达式
+算术加号，和正则表达式，和参数选项前面关闭选项
()可以包含命令组
~表示当前home目录
~+当前工作目录
~-之前工作目录
=~正则表达式匹配
^行首
&0标准输入
&1标准输出
&2标准错误输出
local用于函数，或者代码块
{}大括号扩展 
cat {file,file2,file3}>>12.txt 相当于 cat file file2 file3>>11.txt
cp file.{txt,mydir};相当于cp file.txt file.mydir
backfilename=${1:backup}等价于$1:$backup如果设置了$1就不用$backup
