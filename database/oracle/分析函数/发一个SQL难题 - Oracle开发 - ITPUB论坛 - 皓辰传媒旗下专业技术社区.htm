发一个SQL难题

一个表  flower(planet, flowername)
key是(planet, flowername)

求那些长着所有其他星球上所长有的花的星球的名字。

有人给出如下解答：
select distinct planet
from flower outer
where not exists(
((select distinct flowername from flower b where outer.planet=b.planet) minus

(select distinct flowername from flower c where c.planet <>outer.planet))
union
((select distinct flowername from flower b where outer.planet<>b.planet) minus

(select distinct flowername from flower c where c.planet =outer.planet))
)
or
exists(
(select distinct flowername from flower b where outer.planet=b.planet) minus

(select distinct flowername from flower c where c.planet <>outer.planet)
)
;

有没有更简单点的？
